以下列举了开发者以往被常问到的一些问题，使用时出现问题的用户，请先参照这篇文章自行解决。<br/>
如果依然无法解决你的问题，请前往 [此 issue](https://github.com/hxh19950701/WebViewTvLive/issues/94) 和其他用户一起讨论。<br/>
有过自行解决问题经验的用户，欢迎在 Issues 区留下您的解决方案，来帮助其他用户。<br/>

**Q: 在我的设备上无法自动全屏？**<br/>
A: 由于应用合规问题，本 App 从 2.0 版开始不再默认自动全屏。需要全屏的用户，可以去设置内手动开启。按返回键打开左侧边栏，进入设置，向下翻页，将“自动全屏播放”选项打开即可。<br/>
<br/>

**Q: 打开了自动全屏依然不能自动全屏？**<br/>
A: 按返回键打开左侧边栏，进入设置，将“全屏兼容模式”选项打开即可。<br/>
<br/>

**Q: 提示这台设备 WebView 版本过低，怎么办？**<br/>
请尝试以下任意操作：1. 安装使用 X5 内核。2. 尝试 WebView 电视火狐版。3.尝试升级系统内置 WebView（自行查阅资料）。4. 如果系统 WebView 是 AW，可以直接将 Google WebView 当作普通 App 安装到设备上，最新版 App 会尝试使用它。<br/>
<br/>

**Q: 我如何安装使用 X5 内核？**<br/>
A: 按返回键打开左侧边栏，进入设置，将“WebView 实现”改为 X5，然后进入“TBS 调试界面”，会弹窗询问是否需要安装 X5 内核。<br/>
<br/>

**Q: 一些网站需要登录才能观看/获取更高清晰度，怎么办？**<br/>
A: 点击屏幕右侧或按菜单键打开右侧边栏，点击地球图标可以打开源网页，在这个界面你可以操作源网页来进行登录操作。<br/>
<br/>

**Q: 在 TBS 调试界面“安装线上内核”老是失败？**<br/>
A: 可能刚好遇到了官方 X5 内核下载限流时间段，请换个时间段重试。参见[《关于官网X5内核SDK加载不稳定问题说明》](https://doc.weixin.qq.com/doc/w3_AGoAtwbdAFwlo0hmqkbTl6p19tCOV)。另外某些去广告插件也有可能导致下载失败。<br/>
<br/>

**Q: 应用内显示的 AW，GW，X5，GCK 是什么意思？**<br/>
| 代号   | 名称            | 解释                               | 
| ------ | --------------- | ---------------------------------- | 
| AW     | Android WebView | Android 原生 WebView，不能独立更新|
| GW     | Google WebView  | Google 提供的 WebView              |
| HWW    | Huawei WebView  | Huawei 提供的 WebView              |
| AMW    | Amazon WebView  | Amazon 提供的 WebView              | 
| CHROME | Google Chrome   | Chrome 作为 WebView                |
| UKN    | 未知 WebView    | 系统 WebView 工作不正常            |
| X5     | 腾讯 X5         | X5 WebView                         |
| GCK     | Gecko View         | Gecko 内核，火狐版本                        |
<br/>

**Q: 在我的设备上安装 X5 内核会闪退？**<br/>
A: 安装 X5 内核需要内置 SD 卡（/sdcard）分区。如果你的设备没有这个分区（如部分海信的设备），请检查你的设备有没有“将U盘作为外部存储”类似的选项。也可以尝试 WebView 电视火狐版。<br/>
<br/>

**Q: 使用 X5 内核后，软件运行异常（花屏，闪退，按键失效，卡死等）？**<br/>
A: X5 内核对你的设备支持不好，还是切换回系统 WebView 吧。也可以尝试 WebView 电视火狐版。<br/>
<br/>

**Q: 在魔百盒上安装 X5 内核后重启，软件依然使用的是系统 WebView，没有切换过来？**<br/>
A: 魔百盒的第三方固件太多，质量参差不齐。出现这种情况说明固件有问题，你可以刷其他固件来解决这个问题。<br/>
<br/>

**Q: 访问央视网提示“您当前的浏览器不支持视频播放，请升级浏览器或更换设备（如果是360、 QQ 或搜狗浏览器，须使用极速模式观看）”？**<br/>
A: 系统 WebView 版本太低，升级或使用 X5 内核即可解决。<br/>
<br/>

**Q: 我的设备显示 WebView 版本低，切到 X5 内核又会闪退，花屏，卡死等，怎么办？**<br/>
A: 尝试 WebView 电视火狐版，或者尝试升级系统内置 WebView（自行查阅资料）。<br/>
<br/>

**Q: 尝试了以上所有的办法，在我的设备上依然无法使用？**<br/>
A: 请去这里反馈。https://github.com/hxh19950701/WebViewTvLive/issues/94<br/>
<br/>

**Q: 换台速度特别慢，能不能优化？**<br/>
A: 应用的工作原理决定了换台速度不会快。换台速度取决于设备的性能，无法优化。对于低性能的设备，在设置内打开“无图模式”，或许可以得到些许的性能提升。<br/>
<br/>

**Q: 安装时提示“软件包解析失败”？**<br/>
A: 因为你的设备系统是很老的 4.x 版本，建议还是花点小钱升级一下。<br/>
<br/>

**Q: 有没有适配 Android 5.0 以下设备的打算？**<br/>
A: 没有这个打算。这些设备的性能通常都不高，WebView 版本也比较老而且无法升级，即便适配了，也无法使用。<br/>
<br/>

**Q: 建议提供 1080P 的源？**<br/>
A: 软件内的源都是从互联网上收集而来。电视台官网不提供 1080P 的清晰度，开发者也无能为力。不过你可以切换成其他源，也许有更清晰的。<br/>
<br/>

**Q: 应用运行一段时间后闪退？**<br/>
A: 请确保你使用的是应用的官方的最新版本。另外有些设备性能不高，也无法支持本应用长时间运行。<br/>
<br/>

**Q: 这个项目和电视家有什么关系，图标怎么那么像？**<br/>
A: 没有任何关系，只是使用了电视家的图标。<br/>

# WebViewTV 操作指南

*本文使用 Ai 生成*

WebViewTV 完美适配搭载 Android 系统的 **手机、平板、电视（TV）、车机、掌机** 等多种设备。

## 1. 触摸屏操作 (手机/平板/车机/掌机)

针对触屏设备，屏幕被划分为不同的交互区域：

| 动作 | 交互反馈 | 说明 |
| :--- | :--- | :--- |
| **点击左侧 2/3 区域** | **频道列表** | 快速浏览并切换电视频道 |
| **点击右侧 1/3 区域** | **频道设置** | 调整当前频道的画面比例，换源等 |
| **长按屏幕** | **退出/设置中心** | 弹出退出确认对话框或进入全局设置入口 |
| **上下滑动** | **功能调节** | 默认可调节 **亮度/音量/换台** (可在“设置”中自定义滑动逻辑) |

---

## 2. 遥控器操作 (电视/盒子/投影仪)

针对标准安卓电视遥控器进行了深度优化：

| 按键 | 对应功能 |
| :--- | :--- |
| **OK (确认) 键** | 呼出 **频道列表** |
| **MENU (菜单) 键** | 呼出 **频道设置** |
| **长按 OK 键** | 呼出 **频道设置** (适用于无菜单键的遥控器) |
| **返回键** | 弹出 **退出确认/设置入口** |

---

## 3. 手柄操作 (游戏掌机/外接手柄)

针对具有实体按键的掌机 or 蓝牙手柄玩家：

| 按键 | 对应功能 |
| :--- | :--- |
| **A 键** | 呼出 **频道列表** |
| **X 键** | 呼出 **频道设置** |
| **B 键** | 弹出 **退出确认/设置入口** |


---
*提示：如需修改交互逻辑，请在“设置”菜单中查找“操控设置”项。*
